Movies API documentation

Endpoints

* You have to register a new user first with a post request to "/users" then login with a post request to "/login", in order to use the API

  1. Description Return a list of ALL movies to the user
    The endpoint URL https://cfmovieapp2.azurewebsites.net//movies
    Method GET
    Query parameters -
    request data format -
    response data format JSON
  2. Description Return a list of ALL users to the user
    The endpoint URL https://cfmovieapp2.azurewebsites.net//users
    Method GET
    Query parameters -
    request data format -
    response data format JSON
  3. Description Return data (description, genre, director, image URL, whether it’s featured or not) about a single movie by title to the user
    The endpoint URL https://cfmovieapp2.azurewebsites.net//movies/:title
    Method GET
    Query parameters movie title
    request data format -
    response data format JSON
  4. Description Return user data by username to the user
    The endpoint URL https://cfmovieapp2.azurewebsites.net//user/:username
    Method GET
    Query parameters username
    request data format -
    response data format JSON
  5. Description Return data about a genre (description) by name/title (e.g., “Thriller”)
    The endpoint URL https://cfmovieapp2.azurewebsites.net//movies/genre/:GenreTitle
    Method GET
    Query parameters GenreTitle
    request data format -
    response data format String
  6. Description Return data about a director (bio, birth year, death year) by name
    The endpoint URL https://cfmovieapp2.azurewebsites.net//movies/director/:DirectorName
    Method GET
    Query parameters DirectorName
    request data format -
    response data format JSON
  7. Description Allow new users to register
    The endpoint URL https://cfmovieapp2.azurewebsites.net//users
    Method POST
    Query parameters -
    request data format { "UserName" : "user", "Password": "pass", "Email": "user@use.mo", "Birthday": "yyyy-mm-dd" }
    response data format JSON
  8. Description Allow users to update their user info (username)
    The endpoint URL https://cfmovieapp2.azurewebsites.net//users/:UserName
    Method PUT
    Query parameters user name
    request data format { "UserName" : "user01", "Password": "pass", "Email": "user@use.mo", "Birthday": "yyyy-mm-dd" }
    response data format JSON
  9. Description Allow users to add a movie to their list of favorites
    The endpoint URL https://cfmovieapp2.azurewebsites.net//users/:UserName/:movieID
    Method POST
    Query parameters -
    request data format -
    response data format JSON
  10. Description Allow users to remove a movie from their list of favorites
    The endpoint URL https://cfmovieapp2.azurewebsites.net//users/:UserName/:movieID
    Method DELETE
    Query parameters -
    request data format -
    response data format JSON
  11. Description Allow existing users to deregister
    The endpoint URL https://cfmovieapp2.azurewebsites.net//users/:UserName
    Method DELETE
    Query parameters -
    request data format -
    response data format String
