Endpoints

  1. Description Return a list of ALL movies to the user
    The endpoint URL localhost:8080/movies
    Method GET
    Query parameters -
    request data format -
    response data format JSON
  2. Description Return data (description, genre, director, image URL, whether it’s featured or not) about a single movie by title to the user
    The endpoint URL localhost:8080/movies/:title
    Method GET
    Query parameters movie title
    request data format -
    response data format JSON
  3. Return data about a genre (description) by name/title (e.g., “Thriller”)--- no data available
  4. Return data about a director (bio, birth year, death year) by name--- no data available
  5. Description Allow new users to register
    The endpoint URL localhost:8080/users
    Method POST
    Query parameters -
    request data format JSON , should have name and favoriteMovies properties
    response data format JSON
  6. Description Allow users to update their user info (username)
    The endpoint URL localhost:8080/users/:name
    Method PUT
    Query parameters user name
    request data format JSON , should have name and favoriteMovies properties
    response data format JSON
  7. Description Allow users to add a movie to their list of favorites
    The endpoint URL localhost:8080/users/:id/:movieTitle
    Method POST
    Query parameters -
    request data format -
    response data format JSON
  8. Description Allow users to remove a movie from their list of favorites
